css-padding-margins
====================

really basic less / css utility for adding padding and/or margins to things using classes

##The gist

a three-letter acronym class can be added to any block element to easy add either padding or margin

###Type
p -> padding
m -> magin

###Position
a -> all
t -> top
b -> bottom
l -> left
r -> right
h -> horizontal
v -> vertical

###Size
l -> large
m -> medium
s -> small
n -> none
